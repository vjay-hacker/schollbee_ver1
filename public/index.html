SchoolBee Core Engine API

The centralized, secure multi-tenant platform backing SchoolBee apps. Engineered with Supabase, Redis, and Row-Level Security for next-generation school logistics and communication.

⚡ Real-time Engine

Built-in integration for bus GPS logs, notifications dispatch, and instant parent-teacher messaging powered by Supabase and Redis caching layers.

🔒 Isolated Tenancy

Strict schema isolation and PostgreSQL Row-Level Security (RLS) policies keep school, teacher, student, and parent data securely separated under a unified API structure.

📈 Enterprise Metrics

Pre-instrumented with Prometheus metrics covering request durations, database performance, AI workloads, and tenant segmentations ready for Grafana scraping.

Core API Endpoints Reference

Method Endpoint Description
GET /api/v1/health Validate platform uptime, current node status, and service health check metrics.
POST /api/v1/auth/login Authenticate user credentials and issue secure JSON Web Token for school contexts.
GET /api/v1/students Fetch current school student registers, parent associations, and log information.
POST /api/v1/attendance Record daily attendance, configure custom registers, and submit class states.
POST /api/v1/transport/trips Trigger GPS routes, log passenger details, and track live transport coordinates.
POST /api/v1/ai/chat Query the platform's AI Assistant for natural language summaries of school activities.
GET /metrics Retrieve raw metrics endpoint output formatted specifically for Prometheus scraping.
